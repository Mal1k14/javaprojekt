# 🥊 Sports Groups Manager

## Система управления спортивными группами и анализом характеристик спортсменов

### 📌 О проекте

**Sports Groups Manager** — это полноценное веб-приложение для управления спортивными секциями, позволяющее тренерам и администраторам вести учёт спортсменов, анализировать их физические характеристики и отслеживать прогресс группы.

Проект разработан с использованием современных технологий Java-экосистемы и демонстрирует навыки работы с Spring Boot, базами данных, REST API и фронтенд-разработкой.

---

### 🚀 Функциональные возможности

#### 👥 Управление спортсменами
- Добавление, редактирование и удаление спортсменов
- Указание имени и даты рождения
- Привязка спортсмена к спортивной группе
- Загрузка фотографий спортсменов

#### ️ Управление группами
- Создание спортивных групп (бокс, борьба, плавание и т.д.)
- Описание группы и дата создания
- Просмотр состава группы

####  Система характеристик
- Гибкая система настраиваемых характеристик (сила, скорость, выносливость и др.)
- Привязка характеристик к конкретным группам
- Оценка каждого спортсмена по всем характеристикам (шкала 0-100)
- Визуализация данных в виде таблиц и диаграмм
- ####  Статистика и аналитика
- Детальная статистика по каждому спортсмену
- Сравнение показателей внутри группы
- Процентное соотношение характеристик
- Наглядное отображение прогресса

###  Технологический стек

| Категория | Технологии |
|-----------|-----------|
| **Backend** | Java 26, Spring Boot 4.0.6 |
| **База данных** | H2 Database, JPA/Hibernate |
| **Безопасность** | Spring Security, JWT, BCrypt |
| **Frontend** | HTML5, CSS3, JavaScript |
| **Сборка** | Maven |
| **IDE** | IntelliJ IDEA |
